      Where do I get started?

      1. Click on the Task List tab to add tasks. Specify task name and estimated pomo #.
      2. Click on the task you would like to work on.
      3. Start Timer!

      What is a Pomodoro Timer?

      A Pomodoro timer is a time-management technique. One Pomodoro is a 25 min work session followed by a 5 min break. You can change the length of a pomodoro work session in the settings tab. On the 4th Pomodoro, there is a longer 15 min break. This technique keeps energy and motivation levels high throughout the work schedule.

      What is the TaskList?

      This is the work schedule for the Pomodoro. Here we can add and remove tasks to do. In order to add a task, you only need to specify task name and estimated pomo #. Once all tasks are added, select one to start with! You can drag and drop tasks to align with whatever priority you want.

      How to see completed tasks?

      Visit the statistics tab on the page to view a list of completed tasks and other stats.

      How to customize the timer?

      Visit the settings tab to configure the timer to your liking.

      Why are some functions disabled while the timer is running?

      To ensure max productivity, we disable some functions so that you don't get distracted. If you complete a task while the timer is running, the highest priority task will be assigned to you, no need to access the task list to change tasks.
