      Where do I get started?

      1. Click on the Task List tab to add tasks. Specify task name and estimated pomo #.
      2. Click on the task you would like to work on.
      3. Start Timer!

      What is a Pomodoro Timer?

      A Pomodoro timer is a time-management technique. One Pomodoro or "Pomo" is a 25 min work session followed by a 5 min break. You can change the length of a pomo work session in the settings tab. On the 4th pomo, there is a longer 15 min break. This technique keeps energy and motivation levels high throughout the work schedule.

      How to I pause the timer?

      Part of the Pomodoro technique is that you cannot pause the timer once you start it. This helps keep maximum productivity for a full 25 minutes. If you find that you cannot finish the timer right now, press the stop button and it will count this pomo session as "failed." If you finish a task early, press the done button instead. (More on this below)

      What if I finish a task before the full 25 minutes are over?

      If you finish a task before the timer goes off, press the done button (checkmark icon) to the right of the task name on the timer/home screen. This will mark this task as complete and will automatically transition to the top task in the Tasklist for you to work on for the rest of the time that is left during this pomo.

      What is the TaskList?

      This is the work schedule for the Pomodoro. It is the 2nd icon in the taskbar. Here we can add and remove tasks to do. In order to add a task, you only need to specify task name and estimated pomo #. Once all tasks are added, select one to start with! You can drag and drop tasks to align with whatever priority you want.

      Where can I see my completed tasks?

      Visit the statistics tab (middle icon in the taskbar) on the page to view a list of completed tasks and other stats.

      How to customize the timer?

      Visit the settings tab (rightmost icon in the taskbar) to configure the timer to your liking.

      Why are some functions disabled while the timer is running?

      To ensure max productivity, we disable some functions so that you don't get distracted. If you complete a task while the timer is running, the highest priority task (first one in the list) will be assigned to you, no need to access the task list to change tasks.

      Why did the background color of the page change without my pressing any buttons?

      The background color of the webpage is set to transition to different colors based on the mode you are currently on. While the pomo timer is running, the background is green. When the timer runs out and switches to a short break, the background automatically changes to purple to help visually indicate that time is up. When switching into a long break, the background changes to blue.
